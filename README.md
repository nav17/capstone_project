# Capstone Project

An ETL & Streamlit Project.

### Dataset

https://api.tfl.gov.uk/

## Installation

### Prerequisites

Requires Python 3.13 or above

### Setup

To setup this project:

```
pip install -r requirements-setup.txt
pip install -e .
```


To run (during development):

```
python3 -m scripts.run_etl dev
```
